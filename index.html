Catat Uang — Aplikasi Pencatatan Keuangan

Terakhir diperbarui: 7 Maret 2026

👋 Pendahuluan

Kebijakan privasi ini menjelaskan bagaimana aplikasi Catat Uang ("kami", "aplikasi") mengumpulkan, menggunakan, dan melindungi informasi Anda saat menggunakan aplikasi kami.

✅ Catat Uang adalah aplikasi offline-first. Semua data keuangan Anda disimpan secara lokal di perangkat Anda dan tidak pernah dikirim ke server kami.

📊 Data yang Dikumpulkan

Data yang Disimpan Secara Lokal (di perangkat Anda)

  • Catatan transaksi keuangan (pemasukan dan pengeluaran)
  • Data anggaran (budget) bulanan
  • Daftar belanja
  • Informasi dompet/wallet
  • Preferensi pengaturan aplikasi (tema, PIN)

✅ Data di atas tidak pernah dikirim ke server eksternal. Semua tersimpan hanya di perangkat Anda.

Data yang Dikumpulkan oleh Layanan Pihak Ketiga

Aplikasi ini menggunakan layanan pihak ketiga yang mungkin mengumpulkan informasi:

Layanan Data yang Dikumpulkan Tujuan
Google AdMob Identifier iklan, data perangkat Menampilkan iklan
Google Speech Engine Audio (hanya saat fitur suara aktif) Konversi suara ke teks (input transaksi)
Sistem Biometrik OS Sidik Jari / Wajah Autentikasi. Aplikasi TIDAK PERNAH mengakses atau menyimpan data biometrik Anda.

🔐 Izin Aplikasi

Aplikasi meminta izin berikut dan hanya digunakan untuk tujuan yang dijelaskan:

Izin Tujuan
RECORD_AUDIO Fitur input transaksi menggunakan suara (voice input). Hanya aktif saat pengguna menekan tombol mikrofon.
USE_BIOMETRIC / FINGERPRINT Menggunakan sensor biometrik perangkat untuk mengamankan akses aplikasi.
INTERNET Menampilkan iklan (Google AdMob).
BLUETOOTH Koneksi dengan printer Bluetooth untuk mencetak laporan.

🛡️ Keamanan & Cadangan Data

  • Penyimpanan Lokal: Semua data keuangan disimpan lokal di perangkat Anda menggunakan database SQLite.
  • Keamanan Akses: Aplikasi menyediakan fitur PIN Lock dan Biometrik. Data biometrik (sidik jari/wajah) dikelola sepenuhnya oleh sistem operasi perangkat Anda; aplikasi kami tidak memiliki akses ke data tersebut.
  • Backup Terenkripsi (AES-256): Anda dapat mengekspor data dalam format file .tc yang dikunci dengan enkripsi AES-256 menggunakan password pilihan Anda.

  • ⚠️ PENTING: Password backup tidak disimpan di mana pun. Jika Anda lupa password backup, data dalam file tersebut TIDAK BISA dipulihkan.

  • Tanpa Sinkronisasi Otomatis: Kami tidak menyinkronkan data Anda ke server awan (cloud) secara otomatis. Backup hanya dibuat dan dibagikan secara manual oleh Anda.

📢 Iklan

Aplikasi ini menampilkan iklan melalui Google AdMob untuk mendukung pengembangan aplikasi. Google AdMob mungkin menggunakan identifier perangkat dan cookies untuk menampilkan iklan yang relevan.

Untuk informasi lebih lanjut tentang bagaimana Google menggunakan data Anda, silakan baca Kebijakan Privasi Google.

👶 Privasi Anak-Anak

Aplikasi ini tidak dirancang untuk anak-anak di bawah usia 13 tahun. Kami tidak secara sadar mengumpulkan informasi pribadi dari anak-anak di bawah 13 tahun.
